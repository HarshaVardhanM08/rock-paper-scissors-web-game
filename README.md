# 🎮 Rock Paper Scissors – Modern Interactive Web Game

> A fully responsive, animated, and interactive Rock Paper Scissors web game built using HTML5, CSS3, and Vanilla JavaScript.

---

## 📌 Project Overview

This project is a modern and visually enhanced implementation of the classic **Rock Paper Scissors** game. The goal was not just to recreate the traditional gameplay, but to elevate it using smooth animations, engaging UI design, responsive layout techniques, and structured JavaScript logic.

Unlike basic implementations, this version focuses on:

- 🎨 Modern and attractive user interface
- ⚡ Smooth animation effects and transitions
- 📊 Real-time scoreboard tracking
- ⌨️ Keyboard shortcut support
- 📱 Fully responsive design
- 🧠 Clean and maintainable JavaScript logic

The project demonstrates strong front-end fundamentals and showcases the ability to combine design with functionality.

## 🧠 Game Logic Explanation

1. User selects Rock, Paper, or Scissors.
2. Computer randomly selects one choice.
3. Countdown animation plays.
4. Both choices are revealed.
5. Winner is determined using rule mapping:
   - Rock beats Scissors
   - Paper beats Rock
   - Scissors beats Paper
6. Scoreboard updates dynamically.
7. Buttons are temporarily disabled to prevent multiple clicks.

This structure ensures smooth gameplay and prevents user interaction conflicts.
