# 🎮 Rock Paper Scissors – Modern Interactive Web Game

🔗 **Live Demo:**

## 📌 Project Overview

This project is a modern and visually enhanced implementation of the classic **Rock Paper Scissors** game. The goal was not just to recreate the traditional gameplay, but to elevate it using smooth animations, engaging UI design, responsive layout techniques, and well-structured JavaScript logic.

Unlike basic implementations, this version focuses on delivering a visually appealing user experience combined with clean code architecture.

Key highlights include:

- 🎨 Modern and attractive user interface  
- ⚡ Smooth animation effects and transitions  
- 📊 Real-time scoreboard tracking  
- ⌨️ Keyboard shortcut support  
- 📱 Fully responsive design  
- 🧠 Clean and maintainable JavaScript logic  

This project demonstrates strong front-end fundamentals and the ability to combine design aesthetics with functional logic.

### 🎯 Core Gameplay

- Accurate win/lose/draw logic

## 🧠 Game Logic Explanation

1. User selects Rock, Paper, or Scissors.  
2. Computer randomly selects one option.  
3. A countdown animation plays.  
4. Both choices are revealed with animation.  
5. Winner is determined using rule mapping:
   - Rock beats Scissors  
   - Paper beats Rock  
   - Scissors beats Paper  
6. Scoreboard updates dynamically.  
7. Buttons are temporarily disabled during gameplay to prevent multiple clicks.  

This ensures smooth interaction flow and prevents logical conflicts.
